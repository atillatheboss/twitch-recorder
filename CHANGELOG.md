# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project follows [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Planned
- WebSocket support for real-time updates
- Playlist feature for multiple channels
- Quality settings (480p, 720p, 1080p)
- Automatic conversion to different formats
- Webhook notifications for recording events
- API documentation (Swagger/OpenAPI)

## [1.0.0] - 2026-03-04

### Added
- ✨ **UI Redesign** with modern design and animations
- 🎥 Direct stream recording from Twitch
- ⏰ Time-based scheduling (immediate, delayed, custom)
- 📊 Real-time system monitoring (CPU, RAM, Disk, storage)
- 🌍 Full multilingual support (German & English)
- 📺 Live preview with HLS player
- 🎨 Responsive design for all screen sizes
- 🐳 Docker & Docker Compose support
- 📁 Recording management (upload, download, delete)
- 📝 Comprehensive documentation and README

### Known Limitations
- Only Twitch.tv supported (no YouTube, etc.)
- No built-in authentication in standard configuration
- Maximum 1080p recordings (Twitch API limit)
